# 🏢 Mietspiegel-Rechner Frankfurt am Main 2026

Ein inoffizieller, interaktiver **Mietspiegel-Rechner für Frankfurt am Main (Stand 2026)**. Das Tool ermöglicht eine automatisierte Lagenermittlung, visualisiert Wohnlagen auf einer Karte und vergleicht diese mit den Mietspiegeln 2024 und 2022.

🌐 **Zur Live-Webseite:** [https://mietspiegelffm.github.io/mietspiegelffm/](https://mietspiegelffm.github.io/mietspiegelffm/)

---

## 🚀 Funktionen

* **🗺️ Lagenermittlung:** Automatische Zuordnung von Adressen zur passenden Lagekategorie (1–10) für 2026.
* **📊 Mietspiegel-Vergleich:** Direkter Vergleich der Vergleichsmiete für 2026, 2024 und 2022.
* **📱 Responsive Design:** Optimiert für mobile Geräte und Desktops.

---

## 🛠️ Technologien

Das Projekt basiert auf Standard-Webtechnologien ohne Frameworks oder Build-Tools:
* **HTML5** – Struktur und semantische Inhalte
* **CSS3** – Responsives Layout und Design
* **JavaScript (Vanilla)** – Berechnungslogik und Karten-Interaktion

## 💻 Installation & Lokaler Start

### Voraussetzungen
* **Node.js** (für den lokalen Server-Befehl `npx`)

### Ausführung
Führe folgende Befehle im Terminal aus:

```bash
git clone https://github.com
cd mietspiegelffm
npx serve .
```
